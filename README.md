Organizador Documental con Inteligencia Artificial | Proyecto Personal

Desarrollé un script de automatización en Python para clasificar y organizar documentos (.txt, .pdf) en tiempo real mediante la monitorización de directorios del sistema operativo.

Integré la API de Google Gemini (GenAI) para analizar semánticamente el texto extraído de los archivos, utilizando técnicas de Prompt Engineering para categorizarlos por su contexto.

Implementé librerías como watchdog para la gestión asíncrona de eventos y PyPDF2 para la extracción de texto, creando un flujo de trabajo autónomo y libre de intervención manual.

Tecnologías: Python, API REST (Google GenAI), Watchdog, PyPDF2, Prompt Engineering.
